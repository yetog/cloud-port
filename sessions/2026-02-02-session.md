## Quick Reference

### Start a stopped testing app:
```bash
docker start <container-name>
```

### Rebuild and redeploy an app:
```bash
cd /var/www/zaylegend/apps/testing/<app-name>
npm run build
docker build -f Dockerfile.simple -t <app-name> .
docker stop <app-name> && docker rm <app-name>
docker run -d --name <app-name> -p <port>:80 --restart unless-stopped <app-name>
```

### Test all apps:
```bash
for port in 3010 3012 3013 3014 3015 3016 3017; do
  curl -s -o /dev/null -w "%{http_code} - $port\n" http://localhost:$port/
done
```
