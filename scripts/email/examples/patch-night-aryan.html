THE CONCEPT: TWO POSSIBLE TRACKS

TRACK A: "VIBE CREATOR" (Simpler, Safer)

A tool that generates audio + matching visuals in one click.

How it works:

  1. User selects: Mood → Energy → Genre
  2. System generates: 15-30 sec audio clip (via 11Labs API)
  3. System creates: Matching visual loop (colors, motion, particles)
  4. Output: Downloadable video clip (shareable on social)

TRACK B: "GESTURE VISUALIZER" (Complex, Higher Impact)

A live instrument controlled by hand movements.

How it works:

  1. Camera tracks hand positions (MediaPipe)
  2. Hand position controls audio effects (filter, reverb, delay)
  3. Visuals react to both hands AND audio simultaneously
  4. Performance mode: "I'm playing the music with my hands"

⭐ MY RECOMMENDATION: HYBRID APPROACH

Build Track A as the base, with ONE Track B feature.

Core product: Generate audio + visual loop (reliable)
Magic moment: Raise hand to add reverb/echo effect (wow factor)
Fallback: If gesture fails, vibe creator still works

✓ Reliable demo for the showcase
